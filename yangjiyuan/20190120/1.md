#Codeforces #533 div2 B. Zuhair and Strings

##题解
给你一个长度为n的string,求长度为k的只含一种字符的不重叠子串的最大出现次数。
模拟一下就ok。

```
#include<cstdio>
using namespace std;
int n,k,s[30],ans,tot;
char a,past;
int main()
{
	scanf("%d%d",&n,&k);
	for(int i=1;i<=n;i++)
	{
		cin>>a;
		if(past==a)	tot++;
		else past=a,tot=1;
		if(tot==k) tot=0,s[a-96]++;
	}
	for(int i=1;i<=26;i++) ans=max(s[i],ans);
	printf("%d",ans);
	return  0;
}
```
